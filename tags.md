# Rating

在上面的tags中拼接rating，以空格做间隔。

例如 `eyjafjalla_(arknights) rating:s`

* `rating:g`：general，完全安全的等级
* `rating:s`：Sensitive，没有明显裸露，但是会带有一定性暗示。
* `rating:q`：会裸露除生殖器以外的部分。例如乳头，屁股等
* `rating:e`：无限制
